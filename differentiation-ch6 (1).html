§2 · Основні правила диференціювання
Chain Rule Ланцюгове правило
\(\dfrac{dy}{dx} = \dfrac{dy}{du} \times \dfrac{du}{dx}\)
Де: \(y = f(g(x))\), поклади \(u = g(x)\).
Для \((f(x))^n\): \(n(f(x))^{n-1}\cdot f'(x)\)
\(y = (\sin x)^3\)
\(u = \sin x,\; \frac{dy}{dx} = 3\sin^2\!x\cos x\)
Product Rule Правило добутку
\((uv)' = u'v + uv'\)
Якщо \(y = u(x)\cdot v(x)\), диференціюємо кожен множник почергово.
\(y = e^x \sin x\)
\(y' = e^x \sin x + e^x \cos x = e^x(\sin x + \cos x)\)
Quotient Rule Правило частки
\(\left(\dfrac{u}{v}\right)' = \dfrac{u'v - uv'}{v^2}\)
Порядок важливий: чисельник \(u'v - uv'\), не навпаки!
\(y = \dfrac{\sin x}{e^x}\)
\(y' = \dfrac{\cos x \cdot e^x - \sin x \cdot e^x}{e^{2x}} = \dfrac{\cos x - \sin x}{e^x}\)
§3 · Дотична та нормаль

Що таке дотична і нормаль?

Дотична (Tangent) — пряма, що торкається кривої в точці \(P(a,\,b)\) і має нахил, що дорівнює значенню похідної в цій точці: \(m_T = \dfrac{dy}{dx}\Big|_{x=a}\).

Нормаль (Normal) — пряма, що проходить через ту ж точку перпендикулярно до дотичної. Якщо нахил дотичної \(m_T\), то нахил нормалі: \(m_N = -\dfrac{1}{m_T}\).

📐
Дотична (Tangent)
Торкається кривої в точці, нахил = f′(a)
\(y - b = m_T(x - a)\)
\(m_T = \left.\dfrac{dy}{dx}\right|_{x=a}\)
01Знайди похідну \(f'(x)\)
02Підстав \(x = a\) → отримай нахил \(m_T\)
03Знайди \(b = f(a)\)
04Запиши рівняння \(y - b = m_T(x - a)\)
📏
Нормаль (Normal)
Перпендикулярна до дотичної в тій же точці
\(y - b = m_N(x - a)\)
\(m_N = -\dfrac{1}{m_T} = -\dfrac{1}{f'(a)}\)
01Знайди \(m_T = f'(a)\) як для дотичної
02Нахил нормалі \(m_N = -\dfrac{1}{m_T}\)
03Знайди \(b = f(a)\)
04Запиши рівняння \(y - b = m_N(x - a)\)

Приклад — Дотична та Нормаль

Знайдемо обидві прямі в одній точці

\(y = x^3 - 2x + 1\) при \(x = 2\)
T Рівняння дотичної при \(x=2\)
01 похідна\(f'(x) = 3x^2 - 2\)
02 нахил\(m_T = f'(2) = 3(4) - 2 = 10\)
03 точка\(b = f(2) = 8 - 4 + 1 = 5\), тобто \(P(2,\,5)\)
04 рівняння\(y - 5 = 10(x - 2)\)
Дотична\(y = 10x - 15\)
N Рівняння нормалі при \(x=2\)
01 з прикладу\(m_T = 10\), точка \(P(2,\,5)\)
02 нахил норм.\(m_N = -\dfrac{1}{10}\)
03 рівняння\(y - 5 = -\dfrac{1}{10}(x - 2)\)
04 спростити\(10y - 50 = -(x - 2)\)
Нормаль\(x + 10y = 52\)

Радіани — завжди!
При диференціюванні тригонометричних функцій кути обов'язково мають бути в радіанах. Формули не працюють у градусах.
📍
Стаціонарні точки
Стаціонарна точка — де похідна дорівнює нулю.
\(\dfrac{dy}{dx} = 0\)
Перевіряй тип точки другою похідною.
🔄
Обернена похідна
Якщо легше записати \(x = f(y)\), то:
\(\dfrac{dy}{dx} = \dfrac{1}{\;\dfrac{dx}{dy}\;}\)
⚡
Chain Rule швидко
Зовнішня → диференціюємо → множимо на похідну внутрішньої.
\((\ln(\cos x))' = \dfrac{-\sin x}{\cos x} = -\tan x\)
Наближення малих кутів (x у радіанах)
\(\sin x \approx x\)
\(\tan x \approx x\)
\(\cos x \approx 1 - \dfrac{x^2}{2}\)
Використовується коли x → 0 (дуже малий кут)
