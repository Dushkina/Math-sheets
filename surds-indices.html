Тема 1 · Negative Bases

Від'ємна основа у степені — правила знаку

(-a)ⁿ
Головні правила
Непарний степіньODD power → NEGATIVE
\((-a)^{\text{odd}} < 0\)
Від'ємна основа у непарному степені дає від'ємний результат
\((-2)^1=-2,\quad(-2)^3=-8,\quad(-3)^3=-27\)
Парний степіньEVEN power → POSITIVE
\((-a)^{\text{even}} > 0\)
Від'ємна основа у парному степені дає додатній результат
\((-2)^2=4,\quad(-2)^4=16,\quad(-5)^2=25\)
Увага!-a² vs (-a)²
\(-5^2 \neq (-5)^2\)
-5² = -(5×5) = -25 · дужок немає → спочатку степінь, потім мінус
(-5)² = (-5)×(-5) = +25 · дужки → мінус всередині
Швидкий тестПарний / Непарний?
Парні: 0, 2, 4, 6, 8, 10 … → результат +
Непарні: 1, 3, 5, 7, 9 … → результат −

Trick: \((-1)^n = +1\) якщо n парне, \(-1\) якщо непарне
Розібрані приклади
1Evaluate \(-5^2\)
порядокНемає дужок → спочатку степінь
крок\(-5^2 = -(5\times5) = -25\)
Відповідь\(-25\)
2Evaluate \((-5)^2\)
дужкиЄ дужки → мінус входить у основу
крок\((-5)^2=(-5)\times(-5)=+25\)
Відповідь\(+25\)
3Evaluate \(-3^3\)
крок\(-3^3=-(3\times3\times3)=-27\)
Відповідь\(-27\)
4Evaluate \((-3)^3\)
тест3 — непарне → результат від'ємний
крок\((-3)^3=(-3)\!\times\!(-3)\!\times\!(-3)=9\times(-3)=-27\)
Відповідь\(-27\)
5Evaluate \((-1)^3\times 2^2\times 3\)
крок 1\((-1)^3=-1\) (непарне)
крок 2\(2^2=4\)
крок 3\(-1\times4\times3=-12\)
Відповідь\(-12\)
6Evaluate \(3^2\times(-5)^3\times(-2)\)
крок 1\(3^2=9\)
крок 2\((-5)^3=-125\) (непарне → −)
крок 3\(9\times(-125)\times(-2)=9\times250=2250\)
Відповідь\(2250\)
❌

Типова помилка

\(-5^2\) ≠ \((-5)^2\) — різні вирази! Без дужок мінус не входить в основу.

✅

Правило підрахунку мінусів

Парна кількість від'ємних множників → +
Непарна кількість → −

Тема 2 · Square & Cube Roots

Квадратні та кубічні корені — таблиця і обчислення

\(\sqrt{\phantom{x}}\) · \(\sqrt[3]{\phantom{x}}\)
Ключові визначення
Square RootКвадратний корінь
\(\sqrt{a}=b \iff b^2=a,\quad b\ge0\)
\(\sqrt{a}\) — число, яке у квадраті дає \(a\)
\(\sqrt{25}=5\) бо \(5^2=25\)
Cube RootКубічний корінь
\(\sqrt[3]{a}=b \iff b^3=a\)
\(\sqrt[3]{a}\) — число, яке у кубі дає \(a\). Може бути від'ємним!
\(\sqrt[3]{-8}=-2\) бо \((-2)^3=-8\)
Perfect SquaresПовні квадрати 1–15
\(1^2=1,\ 2^2=4,\ 3^2=9,\ 4^2=16,\ 5^2=25\)
\(6^2=36,\ 7^2=49,\ 8^2=64,\ 9^2=81,\ 10^2=100\)
\(11^2=121,\ 12^2=144,\ 13^2=169,\ 14^2=196,\ 15^2=225\)
Perfect CubesПовні куби 1–10
\(1^3=1,\ 2^3=8,\ 3^3=27,\ 4^3=64,\ 5^3=125\)
\(6^3=216,\ 7^3=343,\ 8^3=512,\ 9^3=729,\ 10^3=1000\)

Тема 3 · Surds — Operations

Множення, ділення, спрощення, додавання і віднімання сурдів

\(a\sqrt{b}\)
Чотири основні правила
Rule 1Multiplication — Множення
\(\sqrt{a}\times\sqrt{b}=\sqrt{a\times b}\)
Множимо те, що під коренями, об'єднуємо у один корінь
\(\sqrt{5}\times\sqrt{15}=\sqrt{75}=5\sqrt{3}\)
Rule 2Division — Ділення
\(\sqrt{a}\div\sqrt{b}=\sqrt{a\div b}\)
Ділимо те, що під коренями
\(\sqrt{40}\div\sqrt{10}=\sqrt{4}=2\)
Rule 3Simplify — Спрощення
\(\sqrt{a\times b}=\sqrt{a}\times\sqrt{b}\)
Знайди найбільший повний квадрат-дільник, виноси його корінь перед знак кореня
\(\sqrt{12}=\sqrt{4\times3}=2\sqrt{3}\)
Rule 4Addition/Subtraction — Збирання
\(a\sqrt{c}\pm b\sqrt{c}=(a\pm b)\sqrt{c}\)
Збирати можна ТІЛЬКИ однакові сурди. Спочатку спрости кожен корінь
\(2\sqrt{3}+6\sqrt{3}=8\sqrt{3}\)
Алгоритм спрощення \(\sqrt{n}\)
📋

Кроки для спрощення сурду

Крок 1: Знайди найбільший множник числа, що є повним квадратом (4, 9, 16, 25, 36, 49…)
Крок 2: Розбий: \(\sqrt{n}=\sqrt{\text{квадрат}\times\text{решта}}\)
Крок 3: Розділи: \(=\sqrt{\text{квадрат}}\times\sqrt{\text{решта}}\)
Крок 4: Обчисли корінь з квадрата і запиши відповідь: \(k\sqrt{m}\)

Розібрані приклади
1Simplify \(\sqrt{12}+2\sqrt{27}\)
спрости 1\(\sqrt{12}=\sqrt{4\times3}=2\sqrt{3}\)
спрости 2\(\sqrt{27}=\sqrt{9\times3}=3\sqrt{3}\) → \(2\times3\sqrt{3}=6\sqrt{3}\)
збери\(2\sqrt{3}+6\sqrt{3}=8\sqrt{3}\)
Відповідь\(8\sqrt{3}\)
2Find \(\sqrt{5}\times\sqrt{15}\)
правило 1\(\sqrt{5}\times\sqrt{15}=\sqrt{75}\)
спрости\(\sqrt{75}=\sqrt{25\times3}=5\sqrt{3}\)
Відповідь\(5\sqrt{3}\)
3Find \(\sqrt{72}\div\sqrt{2}\)
правило 2\(\sqrt{72}\div\sqrt{2}=\sqrt{72\div2}=\sqrt{36}\)
крок\(\sqrt{36}=6\)
Відповідь\(6\)
4Simplify \(2\sqrt{32}+3\sqrt{2}\)
спрости\(\sqrt{32}=\sqrt{16\times2}=4\sqrt{2}\)
підстав\(2\times4\sqrt{2}+3\sqrt{2}=8\sqrt{2}+3\sqrt{2}\)
збери\(11\sqrt{2}\)
Відповідь\(11\sqrt{2}\)
5Simplify \(7\sqrt{7}-3\sqrt{7}\)
однаковіОбидва мають \(\sqrt{7}\) → можна збирати
крок\((7-3)\sqrt{7}=4\sqrt{7}\)
Відповідь\(4\sqrt{7}\)
6Simplify \(\sqrt{108}+2\sqrt{300}\)
спрости 1\(\sqrt{108}=\sqrt{36\times3}=6\sqrt{3}\)
спрости 2\(\sqrt{300}=\sqrt{100\times3}=10\sqrt{3}\) → \(2\times10\sqrt{3}=20\sqrt{3}\)
збери\(6\sqrt{3}+20\sqrt{3}=26\sqrt{3}\)
Відповідь\(26\sqrt{3}\)
❌

Не можна збирати різні сурди

\(2\sqrt{3}+3\sqrt{7}\neq5\sqrt{10}\) — це НЕ спрощується далі, сурди різні!
