§1 · Compound Angle Formulae (формулы сложных углов)
Compound Angle Formulae
Для любых углов A и B — must memorise!
§ 4.1
sin addition
\(\sin(A+B) = \sin A\cos B + \cos A\sin B\)
sin subtraction
\(\sin(A-B) = \sin A\cos B - \cos A\sin B\)
cos addition
\(\cos(A+B) = \cos A\cos B - \sin A\sin B\)
cos subtraction
\(\cos(A-B) = \cos A\cos B + \sin A\sin B\)
tan addition
\(\tan(A+B) = \dfrac{\tan A + \tan B}{1 - \tan A\tan B}\)
tan subtraction
\(\tan(A-B) = \dfrac{\tan A - \tan B}{1 + \tan A\tan B}\)
💡

Memory trick — знаки!

sin: знак между слагаемыми всегда такой же как в скобках: sin(A+B) → +, sin(A−B) → −
cos: знак противоположный: cos(A+B) → −, cos(A−B) → +

§2 · Double Angle Formulae (формулы двойного угла)
sin double angle
\(\sin 2A = 2\sin A\cos A\)
cos double angle — 3 формы!
\(\cos 2A = \cos^2\!A - \sin^2\!A\)
\(= 2\cos^2\!A - 1\)
\(= 1 - 2\sin^2\!A\)
tan double angle
\(\tan 2A = \dfrac{2\tan A}{1-\tan^2\!A}\)
Полезные преобразования с cos 2A
Часто встречается на экзамене
Key rearrangements
cos² выражен через cos 2A
\(\cos^2\!A = \dfrac{1+\cos 2A}{2}\)
sin² выражен через cos 2A
\(\sin^2\!A = \dfrac{1-\cos 2A}{2}\)
Пример: \(\sqrt{1+\cos 2\theta}\)
\(\sqrt{2\cos^2\!\theta} = \sqrt{2}\,|\cos\theta|\)
Пример: \(\sin^2\!\theta\cos^2\!\theta\)
\(\tfrac{1}{4}(1-\cos 2\theta)(1+\cos 2\theta) = \tfrac{1}{4}\sin^2\!2\theta\)

§3 · R·sin / R·cos Form (гармоническая форма)
Записать в виде \(R\sin(x \pm \alpha)\) или \(R\cos(x \pm \alpha)\)
где \(R > 0\) и \(0 < \alpha < \tfrac{\pi}{2}\)
R-form
R·sin form
\(a\sin x + b\cos x = R\sin(x+\alpha)\)
где \(R\cos\alpha = a\), \(R\sin\alpha = b\)
R·cos form
\(a\cos x - b\sin x = R\cos(x+\alpha)\)
где \(R\cos\alpha = a\), \(R\sin\alpha = b\)
Нахождение R
\(R = \sqrt{a^2 + b^2}\)
Нахождение α
\(\tan\alpha = \dfrac{b}{a} \Rightarrow \alpha = \arctan\!\dfrac{b}{a}\)

Алгоритм записи в R-form

4 шага для выражения \(a\cos x \pm b\sin x\) в виде \(R\sin/\cos\)

4 steps
1
Раскрой формулу
Раскрой \(R\sin(x-\alpha)\) или \(R\cos(x+\alpha)\) по compound angle formula и сравни коэффициенты
2
Найди R
Возведи в квадрат и сложи: R² = a² + b², потім \(R = \sqrt{a^2+b^2}\)
3
Найди α
Раздели одно уравнение на другое: tan α = b/a, потім \(\alpha = \arctan\!\frac{b}{a}\)
4
Запиши ответ
Подставь R и α. Для min/max: R·sin = ±R, поэпоэтому min(f) и max(f) сразу известны

Пример 1 — R·sin form (с урока)

Exam Question: выразить в виде \(R\sin(2x-\alpha)\)

\(f(x)=\sqrt{3}\sin 2x - 3\cos 2x\)
①Находим R
expand\(R\sin(2x-\alpha) = R\sin 2x\cos\alpha - R\cos 2x\sin\alpha\)
compare\(R\cos\alpha = \sqrt{3}\) и \(R\sin\alpha = 3\)
square+add\(R^2\cos^2\!\alpha + R^2\sin^2\!\alpha = 3 + 9 = 12\)
result\(R^2 = 12 \Rightarrow R = \sqrt{12} = 2\sqrt{3}\)
R =\(2\sqrt{3}\)
②Находим α
divide\(\dfrac{R\sin\alpha}{R\cos\alpha} = \dfrac{3}{\sqrt{3}} = \sqrt{3}\)
solve\(\tan\alpha = \sqrt{3} \Rightarrow \alpha = 60° = \dfrac{\pi}{3}\)
Ответ\(f(x) = 2\sqrt{3}\sin\!\left(2x-\dfrac{\pi}{3}\right)\)

Пример 2 — R·cos form (с урока)

Exam Question: выразить в виде \(R\cos(x+\alpha)\)

\(f(x) = 7\cos x - 24\sin x\)
①Находим R
compare\(R\cos\alpha = 7\), \(R\sin\alpha = 24\)
square+add\(R^2 = 49 + 576 = 625\)
R =\(25\)
②Находим α та min g(x)
α\(\tan\alpha = \tfrac{24}{7} \Rightarrow \alpha \approx 1.287\) рад (\(73.7°\))
f(x)\(f(x) = 25\cos(x + 1.287)\)
g(x)\(g(x) = \dfrac{5}{90-3f(2x)} = \dfrac{5}{90-75\cos(2x+1.287)}\)
min gДробь min → знаменатель max → \(\cos = 1\): \(g_\min = \dfrac{5}{90-75} = \dfrac{5}{15} = \dfrac{1}{3}\)
g(min) =\(\dfrac{1}{3}\)
§4 · Решение тригонометрических уравнений
sin x = a−1 ≤ a ≤ 1
\(x = (-1)^k \arcsin a + \pi k,\; k\in\mathbb{Z}\)
или в градусах: \(x = (-1)^k \arcsin a + 180°k\)
Особые случаи
\(\sin x=0\)
\(x=\pi k\)
\(\sin x=1\)
\(x=\frac{\pi}{2}+2\pi k\)
\(\sin x=-1\)
\(x=-\frac{\pi}{2}+2\pi k\)
cos x = a−1 ≤ a ≤ 1
\(x = \pm\arccos a + 2\pi k,\; k\in\mathbb{Z}\)
или в градусах: \(x = \pm\arccos a + 360°k\)
Особые случаи
\(\cos x=0\)
\(x=\frac{\pi}{2}+\pi k\)
\(\cos x=1\)
\(x=2\pi k\)
\(\cos x=-1\)
\(x=\pi+2\pi k\)
tan x = aa ∈ ℝ
\(x = \arctan a + \pi k,\; k\in\mathbb{Z}\)
или в градусах: \(x = \arctan a + 180°k\)
Також: cot x = a
\(\cot x = a \Rightarrow x = \text{arccot}\,a + \pi k\)

Пример 3 — Уравнение с double angle

Solve for \(0 \leq x < 360°\)

\(2\cos 2x = 7\cos x\)
①Подстановка double angle
expand\(2(\cos^2\!x - \sin^2\!x) = 7\cos x\)
sub\(\sin^2\!x = 1-\cos^2\!x\):
simplify\(4\cos^2\!x - 7\cos x - 2 = 0\)
let\(t = \cos x\): \(4t^2 - 7t - 2 = 0\)
D\(D = 49 + 32 = 81 \Rightarrow t = \dfrac{7\pm9}{8}\): \(t = 2\) или \(t = -\tfrac{1}{4}\)
reject\(t=2\) rejected (\(|\cos x|\leq 1\)), поэтому \(\cos x = -\tfrac{1}{4}\)
②Решение для \(0 \leq x < 360°\)
solve\(x = \pm\arccos(-0.25) + 360°k\)
compute\(\arccos(-0.25) \approx 104.5°\)
k=0, +\(x = 104.5°\) ✓
k=1, −\(x = -104.5° + 360° = 255.5°\) ✓
Ответ\(x = 104.5°\) та \(x = 255.5°\)
Графики тригонометрических функций
y = sin x
Range: \(-1 \leq \sin x \leq 1\)
y = cos x
Range: \(-1 \leq \cos x \leq 1\)
y = R·sin(x + α)
Amplitude = R, смещение на α
⚡

Min/Max из R-form — ключевая идея!

Если \(f(x) = R\sin(x+\alpha)\), то max f = R и min f = −R.
Если \(g(x) = \dfrac{k}{A + R\sin(x+\alpha)}\), то чтобы g была минимальной → знаменатель должен быть максимальным → \(\sin = 1\).
Чтобы g была максимальной → знаменатель должен быть минимальным → \(\sin = -1\).
