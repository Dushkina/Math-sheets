§1 · Что такое стационарная точка?

Точка, где градиент кривой равен нулю

Стационарная точка (или turning point) — это точка на кривой, где производная равна нулю. В этой точке функция ни возрастает, ни убывает — она «поворачивает». Стационарные точки бывают максимумом или минимумом.

\(\dfrac{dy}{dx} = 0\)
§2 · Максимум и минимум
Maximum Локальный максимум
Градиент убывает: от +ve → 0 → −ve
Слева от точки: \(f'(x) > 0\)
Справа от точки: \(f'(x) < 0\)
\(f''(x) < 0 \Rightarrow \text{максимум}\)
Minimum Локальный минимум
Градиент возрастает: от −ve → 0 → +ve
Слева от точки: \(f'(x) < 0\)
Справа от точки: \(f'(x) > 0\)
\(f''(x) > 0 \Rightarrow \text{минимум}\)
Формы кривых и их стационарные точки
Квадратичная, a > 0
1 минимум
Квадратичная, a < 0
1 максимум
Кубическая, a > 0
max + min
Кубическая, a < 0
min + max
§3 · Метод знакового графика (Sign Chart)

Алгоритм нахождения стационарных точек

Четыре шага — всегда в таком порядке

4 шага
1
Найди производную
Дифференцируй \(y = f(x)\) → получи y' = f'(x)
2
Приравняй к нулю
Поставь f'(x) = 0 → реши уравнение → найди \(x\)
3
Знак f′(x)
Подставь точку слева и справа от \(x_0\) → определи знак
4
Сделай вывод
+ → − : max. − → + : min. Найди \(y = f(x_0)\)

Знаковый график — как читать

Проверяем знак f′(x) слева и справа от стационарной точки

Справа от x₀
f′ < 0 (−)
Знак меняется + → −: точка \(x_0\) — МАКСИМУМ

f′ < 0 (−)

Справа от x₀
f′ > 0 (+)
Знак меняется − → +: точка \(x_0\) — МИНИМУМ
§4 · Монотонность функции

Таблица монотонности

Функция возрастает там, где f′ > 0; убывает там, где f′ < 0

Условие Поведение f(x) На интервале Знак f′
\(f'(x) > 0\) f(x) возрастает ↑ f ↗ на \((a,\,b)\) \(+\)
\(f'(x) < 0\) f(x) убывает ↓ f ↘ на \((a,\,b)\) \(-\)
\(f'(x) = 0\) Стационарная точка max или min \(0\)
Разобранные примеры

Пример 1 — Квадратичная функция

Находим стационарную точку и определяем её тип

\(y = 2x^2 - 4x + 7\)
1Находим критическую точку
производная\(y' = 4x - 4\)
= 0\(4x - 4 = 0 \Rightarrow x = 1\)
y-координата\(y(1) = 2 - 4 + 7 = 5\)
Точка\((1;\;5)\)
2Определяем тип — знаковый график
слева x=0\(y'(0) = -4 < 0\) → убывает
справа x=2\(y'(2) = 4 > 0\) → возрастает
знак\((-)\to(+)\) — минимум
монотонн.↓ на \((-\infty;\,1)\), ↑ на \((1;\,+\infty)\)
Вывод\(x=1\) — min

Пример 2 — Кубическая функция

Две стационарные точки: max и min

\(y = x^3 - 9x^2 + 3\)
1Находим обе точки
производная\(y' = 3x^2 - 18x\)
= 0\(3x(x-6) = 0\)
точки\(x = 0\) или \(x = 6\)
y-коорд.\(y(0) = 3\),  \(y(6) = -105\)
Точки\((0;\,3)\) и \((6;\,-105)\)
2Тип каждой — знаковый график
x=−2\(y'(-2) = 12+36 > 0\) → +
x=1\(y'(1) = 3-18 = -15 < 0\) → −
x=8\(y'(8) = 192-144 > 0\) → +
знаки\(x=0\): + → − (max);  \(x=6\): − → + (min)
Вывод\(x=0\) — max;  \(x=6\) — min
