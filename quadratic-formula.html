§1 · Формула корней квадратного уравнения

Квадратное уравнение общего вида

Если \(ax^2+bx+c=0\), где \(a\), \(b\), \(c\) — постоянные и \(a \neq 0\), то корни находятся по формуле справа. Сначала всегда вычисляется дискриминант \(D = b^2-4ac\) — он определяет, сколько решений имеет уравнение.

\(x = \dfrac{-b\pm\sqrt{b^2-4ac}}{2a}\)

Дискриминант — формула

Дискриминант обозначается буквой \(D\) (иногда \(\Delta\)) и вычисляется отдельно перед подстановкой в формулу корней.

\(D = b^2-4ac\)
Три случая в зависимости от знака D
D > 0Два корня
\(x_1 = \dfrac{-b+\sqrt{D}}{2a}\)
\(x_2 = \dfrac{-b-\sqrt{D}}{2a}\)
Уравнение имеет два различных действительных корня
D = 0Один корень
\(x = \dfrac{-b}{2a}\)
Уравнение имеет один корень (кратности 2) — вершина параболы касается оси x
D < 0Решений нет
x ∈ ∅
Действительных корней нет, так как \(\sqrt{D}\) не определён для отрицательного числа
§2 · Пошаговый алгоритм решения

5 шагов — всегда в таком порядке

Универсальный метод для любого квадратного уравнения

5 steps
1
Приведи к стандартному виду
Перенеси всё в одну сторону: ax²+bx+c=0. Определи коэффициенты a, b, c
2
Вычисли D
D = b²−4ac. Определи знак D — это решит, сколько будет корней
3
Проверь знак D
D > 0 → 2 корня. D = 0 → 1 корень. D < 0 → корней нет, запиши ответ
4
Подставь в формулу
x = (−b±√D)/2a. Вычисли x₁ и x₂ отдельно
⚠️

Частые ошибки

Не забывай знак минус перед \(b\) в формуле — если \(b\) отрицательный, то \(-b\) станет положительным. Следи за знаком при вычислении \(D = b^2-4ac\): возведение в квадрат всегда даёт положительное число, даже если \(b\) отрицательное. Коэффициент \(a\) никогда не равен нулю — иначе уравнение перестаёт быть квадратным.

§3 · Метод замены — уравнения, сводящиеся к квадратным

Когда применяется замена

Некоторые уравнения не являются квадратными по виду, но содержат повторяющийся блок \(x^2\) или \(x^3\), который можно заменить новой переменной \(t\). После замены получается обычное квадратное уравнение относительно \(t\). Самые частые типы: биквадратные уравнения \(ax^4+bx^2+c=0\) (замена \(t=x^2\)) и уравнения вида \(ax^6+bx^3+c=0\) (замена \(t=x^3\)).

Алгоритм решения методом замены

Для биквадратных уравнений \(ax^4+bx^2+c=0\)

4 шага
1
Сделай замену
Обозначь x² = t (или x³ = t, если степень 6). Получи квадратное уравнение относительно t
2
Реши уравнение для t
Используй формулу корней или разложение на множители — найди t₁ и t₂
3
Вернись к замене
Подставь найденные t обратно: x² = t₁ и x² = t₂
4
Проверь t > 0
Если x² = t, то t не может быть отрицательным! Отбрось отрицательные t
💡

Ключевое правило проверки

Так как \(x^2 = t\), значение \(t\) обязано быть неотрицательным (\(t \geq 0\)), потому что квадрат любого действительного числа не может быть отрицательным. Если при решении уравнения относительно \(t\) получился отрицательный корень, его нужно отбросить — он не даёт действительных значений \(x\). Для каждого положительного \(t\) уравнение \(x^2=t\) даёт два значения: \(x = \pm\sqrt{t}\).

§4 · Разобранные примеры

Пример 1 — обычное квадратное уравнение

Применение формулы корней напрямую

\(x^2 - 6x = -4\)
1Приводим к стандартному виду
перенос\(x^2 - 6x + 4 = 0\)
коэфф.\(a=1,\; b=-6,\; c=4\)
D\(D = (-6)^2 - 4(1)(4) = 36-16 = 20\)
D > 0два корня
2Подставляем в формулу
формула\(x = \dfrac{6\pm\sqrt{20}}{2}\)
упростить\(\sqrt{20}=2\sqrt{5}\), значит \(x = \dfrac{6\pm2\sqrt{5}}{2} = 3\pm\sqrt{5}\)
Ответ\(x = 3+\sqrt{5}\) или \(x = 3-\sqrt{5}\)

Пример 2 — биквадратное уравнение (метод замены)

Замена \(x^2=t\), оба t положительные

\(x^4 - 13x^2 + 36 = 0\)
1Замена и решение для t
замена\(x^2 = t \Rightarrow t^2 - 13t + 36 = 0\)
множителиИщем числа: произведение 36, сумма −13 → \(-9\) и \(-4\)
разложение\((t-9)(t-4)=0\)
t =\(t_1=9,\; t_2=4\) — оба положительные ✓
2Возврат к замене x² = t
t=9\(x^2=9 \Rightarrow x=\pm3\)
t=4\(x^2=4 \Rightarrow x=\pm2\)
Ответ\(x = \pm2,\; \pm3\)

Пример 3 — биквадратное с иррациональными корнями

Замена \(x^2=t\), корни не целые числа

\(x^4 - 6x^2 + 5 = 0\)
1Замена и решение для t
замена\(x^2=t \Rightarrow t^2-6t+5=0\)
множителипроизведение 5, сумма −6 → \(-5\) и \(-1\)
разложение\((t-5)(t-1)=0\)
t =\(t_1=5,\; t_2=1\) — оба положительные ✓
2Возврат к замене x² = t
t=5\(x^2=5 \Rightarrow x=\pm\sqrt{5}\)
t=1\(x^2=1 \Rightarrow x=\pm1\)
Ответ\(x=\pm1,\;\pm\sqrt{5}\)

Пример 4 — уравнение со степенью 6 (замена x³ = t)

Один корень t отрицательный — НЕ отбрасываем, так как x³ может быть отрицательным!

\(x^6 - 7x^3 - 8 = 0\)
1Замена и решение для t
замена\(x^3=t \Rightarrow t^2-7t-8=0\)
множителипроизведение −8, сумма −7 → \(-8\) и \(+1\)
разложение\((t-8)(t+1)=0\)
t =\(t_1=8,\; t_2=-1\)
2Возврат к замене x³ = t
важно!\(x^3=t\) — в отличие от \(x^2\), кубический корень существует и для отрицательных t!
t=8\(x^3=8 \Rightarrow x=2\)
t=−1\(x^3=-1 \Rightarrow x=-1\)
Ответ\(x=2\) или \(x=-1\)
🔑

Разница между заменой x² = t и x³ = t

При замене x² = t значение \(t\) обязано быть \(\geq 0\) — отрицательные \(t\) отбрасываются.
При замене x³ = t такого ограничения нет — кубический корень определён для любого числа, поэтому отрицательные значения \(t\) дают полноценный действительный корень \(x = \sqrt[3]{t}\).
